# Node.js 的模块

> 以下分析以 node.js v10.15.3 源码为基础(node/src/node_version.h)

Node.js 大致分为以下几类：

1. C++ 编写的内置模块(Builtin Module);
2. Javascript 编写的内嵌模块(Native Module);
3. C++ 编写的扩展模块;
4. Javascript 编写的应用模块.
